# Credit Card Fraud Detection Pipeline & Analytics Dashboard

## Project Overview
An end-to-end data analytics and engineering project that simulates a financial fraud monitoring ecosystem. This system extracts transaction data, runs a multi-stage Python ETL pipeline to filter anomalies, and stores records in a structured SQLite database.

>  **Status: Frontend Dashboard Under Active Development**
> The backend Python ETL pipeline and SQLite database configurations are fully functional and committed below. The interactive Power BI security monitoring interface is currently being polished for deployment.

## Tech Stack & Architecture
* **Data Engineering:** Python (Pandas)
* **Database Layer:** SQL, SQLite3
* **Business Intelligence:** Power BI (In Progress)

## Core Features (Engineered Backend)
* **High-Velocity Rules Engine:** Programmed backend validation logic to isolate suspicious burst spending cycles.
* **Schema Design:** Structured custom relational database tables to track user, card, and transaction facts seamlessly.
